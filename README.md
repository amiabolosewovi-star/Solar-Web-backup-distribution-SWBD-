SWBD - Solar Web Backup Distribution
Description

SWBD est une plateforme GreenTech qui permet de connecter des maisons équipées de panneaux solaires dans un réseau intelligent.

La solution redistribue automatiquement l'énergie excédentaire des maisons disposant d'un surplus vers celles confrontées à un déficit énergétique afin de réduire les coupures d'électricité et d'optimiser l'utilisation des énergies renouvelables.

Problématique

Les coupures d'électricité et l'accès limité à l'énergie restent un défi majeur dans plusieurs communautés.

SWBD propose une approche collaborative permettant aux habitations productrices d'énergie solaire de partager leur surplus avec les foyers en manque.
